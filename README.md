# my-word-app

一个基于 React + Vite 的极简英语单词学习应用，支持导入词表、分阶段学习、学习统计与本地持久化。

## 已实现功能

- 多页面路由：首页、导入页、学习页、统计页
- 本地存储：使用 `localStorage` 保存词库、学习进度、统计数据
- 导入功能：
  - 文本粘贴导入（每行 `word meaning`）
  - CSV 文件导入（支持 `word,meaning`，可自动跳过表头）
  - 自动去重、忽略空行和无效行
- 学习流程：
  - 新词先进行认识判断（认识/不认识）
  - 认识后进入拼写练习（提示拼写到完整拼写）
  - 拼写错误可回退修改
  - 发音功能：支持播放当前单词发音
- 统计功能：
  - 今日答题数、学习词汇数、今日学习时长
  - 当月日历式学习热力图
  - 一键重置学习数据（带谨慎提示）

## 本地运行

```bash
npm install
npm run dev
```

默认开发地址（Vite）：`http://localhost:5173`

## 构建与预览

```bash
npm run build
npm run preview
```
